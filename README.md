# Customer Churn Prediction

## Overview
This project predicts customer churn for a business using machine learning. It includes end-to-end workflow from data preprocessing to model evaluation. The goal is to identify customers likely to leave, enabling proactive retention strategies.

## Features
- Exploratory Data Analysis (EDA) with visualizations
- Data cleaning and preprocessing
- Feature engineering for better predictions
- Multiple ML models: Logistic Regression, Random Forest, XGBoost
- Model evaluation with accuracy, precision, recall, F1-score, and ROC-AUC
- Easy-to-use Jupyter/Colab notebooks

## Tech Stack
- **Python** – core language
- **Pandas & NumPy** – data manipulation
- **Matplotlib & Seaborn** – visualization
- **Scikit-learn** – model training and evaluation
- **Google Colab** – interactive notebooks
